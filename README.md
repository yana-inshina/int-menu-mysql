Консольное меню: проверка чисел + MySQL + Excel
Требования
JDK 17+
MySQL 8.x (локальный хост:3306)
Установка БД
В MySQL выполнить:
SOURCE create_db.sql;
При необходимости сменить креды в src/main/java/org/example/Main.java
(константы DB_URL / DB_USER / DB_PASS).
Запуск
Сборка JAR:
mvn -q -DskipTests package
Запуск:
java -jar target/int-menu-mysql-1.0.0-shaded.jar
Проверка пунктов меню
1 — вывести все таблицы из MySQL
(проверка подключения к БД и выбранной схемы)
2 — создать/проверить таблицу int_results
3 — выполнение задачи базового варианта:
   • ввести с клавиатуры несколько значений (через пробел);
   • для каждого значения определить:
     – является ли оно целым числом;
     – если это целое число — проверить чётность;
   • сохранить результат проверки в MySQL (таблица int_results);
   • вывести результаты проверки в консоль.
4 — экспортировать все записи из int_results в файл int_results.xlsx
и вывести данные в консоль.
Примечания
Ввод для пункта 3 — произвольная последовательность токенов через пробел
(например: 10 3.5 abc -4 0).
Если значение нельзя преобразовать к int, оно считается «не целым числом»
и в БД попадает с признаком is_integer = 0 и текстовой пометкой в поле note.
Для целых чисел в БД сохраняются:
оригинальный ввод (original_text),
значение int_value,
признак чётности is_even.
Таблица int_results создаётся либо через create_db.sql, либо через пункт меню 2.
Файл Excel int_results.xlsx создаётся рядом с исполняемым JAR
(или в корне проекта при запуске из IDE).
